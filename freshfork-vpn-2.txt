# Чтобы раздельный туннель (Split Tunnel) заработал корректно, настройте правила в Hiddify:
#
# 1. Настройка категорий (Группы прокси):
#    - Создайте три группы прокси на основе тегов из подписки:
#      • Группа "Roblox-LTE" -> Выберите ноды со значком [Roblox-LTE]
#      • Группа "Gemini-AI"  -> Выберите ноды со значком [Gemini-AI]
#      • Группа "General-YT" -> Выберите ноды со значком [General-YT-TG]
#
# 2. Правила маршрутизации (Routing Rules):
#
#    A. ПРАВИЛО ДЛЯ ROBLOX-LTE:
#       - Тип: Домены (Domain) и Названия приложений (Package Name)
#       - Значения (Домены): roblox.com, www.roblox.com, assetdelivery.roblox.com, rbxcdn.com
#       - Выход (Outbound): Группа прокси "Roblox-LTE"
#       - Примечание: Благодаря маскировке под RU SNI (x5.ru, vk.com, max.ru), ТСПУ операторов 
#         пропускает этот трафик по "белому списку", сохраняя низкий пинг и предотвращая дропы UDP.
#
#    B. ПРАВИЛО ДЛЯ GEMINI-AI:
#       - Тип: Домены (Domain)
#       - Значения: algo.google, bard.google.com, gemini.google.com, generativelanguage.googleapis.com, ai.google.dev
#       - Выход (Outbound): Группа прокси "Gemini-AI"
#       - Примечание: Запросы уйдут через чистые резидентные/VPS пулы Европы без RU SNI,
#         что полностью исключает появление 403 Forbidden со стороны Google Anticheat/WAF.
#
#    C. ПРАВИЛО ПО УМОЛЧАНИЮ (GLOBAL / GENERAL):
#       - Все остальные заблокированные ресурсы (YouTube, Telegram, Instagram, Twitter) направьте
#         на группу прокси "General-YT-TG".
#       - Значения для YouTube/TG: youtube.com, googlevideo.com, ytimg.com, t.me, telegram.org
#       - Выход (Outbound): Группа прокси "General-YT-TG"
